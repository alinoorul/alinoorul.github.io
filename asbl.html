Projects in ASBL:

1. ASBL Lab 0 ($500,000): this machine feeds bacterial and mammalian cells to keep cells alive. ASBL Lab 0 integrates a liquid handler, plate reader, incubator, refrigerator, cell counter, and robotic arm. Use with Python.

2. ASBL Lab 1 ($2.5 million): this machine genetically engineers E. coli K12 cells. It adds plasmid transformation capability to ASBL Lab 0. Support equipment also integrated. This machine outputs a genetically engineered bacterial colony given an input plasmid. Use with Python.

3. ASBL Lab 2 ($100 million): this machine generates therapeutic lead candidates. It can manufacture personalized therapies for patients in clinic. ASBL uses this machine to develop insulin replacements for subtypes of type 2 diabetes. Use with Python.

4. HumanGEMlib is a Python library for human genome-scale modeling. HumanGEM (1) is a systems biology model available online. It contains 12995 reactions in a stochiometric matrix. Using COBRApy, another Python library for flux balance analysis, HumanGEMlib simulates the effects of genetic knockouts on human metabolism reactions using genes identified in type 2 diabetes genome-wide association studies (GWAS). Compared to no-knockout control condition, knockout reaction fluxes differ highlighting the downstream effects of genetic knockouts of genes associated with type 2 diabetes. Subtypes of type 2 diabetes were highlighted from 4-5 signature metabolisms, reported here.

5. Insulin replacement for type 2 diabetes: our early work using HumanGEMlib highlighted subtypes of type 2 diabetes (T2D). Each subtype has a unique metabolic signature, in which certain pathways are dysregulated. Our hypothesis based on these data is that each subtype ought to be treated differently, since different reaction pathways are dysregulated in each subtype of T2D.

6. Matrixims: brain organoids (3) playing The Sims

7. Deconstructed human: organoid (4,5) bioreactors for testing

8. Automated E. coli DNA integration (better than Kytopen)

9. Lasers from proteins. Reverse photosynthesis is emission of narrow wavelength light from a biological protein like chlorophyll (possibly conjugated with quantum dot).

10. Cellular agriculture of wasabi and truffles

11. Lab automation for cultivated meat

12. Microtubules in carbon nanotube applications
