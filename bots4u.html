STDinfo implements privacy-preserving data analytics for medical data. It is a Python script that infers transmission risk from 2 medical test reports. It is applied to sexually transmitted diseases (STDs) to find transmission risk between 2 individuals based on their STD panel test reports while keeping the reports private. Performing data analytics while keeping underlying data private has uses in healthcare data analyses, semiconductor fabrication optimization and privacy in large language models. If you'd like for us to explore these or other applications to implement STDinfo in your organization, we are currently running a free pilot till May 2024.

Automeet is a bot to attend video meetings. In the linked GitHub repository, the script 'chrome.py' keeps track of a timetable to attend and exit Google meetings within stipulated times. Graphical user interface is manipulated using basic computer vision to enter and exit a video conference just as a human would. This code needs significant refactoring to adapt to new scenarios.
