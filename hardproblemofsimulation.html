HARD PROBLEM OF SIMULATION


The hard problem in simulations is distribution of power, processing cycles, and

memory between:

•World generation: creating the “world” i.e., collection of stimuli to allow agent

interaction. The algorithms to create this world fall into this

•World stimuli: providing stimuli as input to agent and letting bot record it

•Self-reflection: algorithms to make the agent self -reflect, i.eread log of self -

data, interaction data

•Recording interactions between bots, and between a bot and itself

Distributing computing power across these tasks is a hard problem
