RESEARCH ENGINE


Get a dex

Get your dex here. Python required.

A dex is an Internet journal to put your research papers on the Internet. Think self-hosted arXiv. Put PDFs of your research papers in a folder. A Python script, dev.py, makes a website from PDFs. You need to know some Python to change website properties.

Run dev.py once. Put PDFs in raw folder. Run dev.py twice. Index.html in the dex folder is your website.

Put this website on GitHub Pages. You have a text-only collection of pages. Each page is your research paper.

This is your own journal freely available on the Internet.

Put out truth because credibility lost is not won back.

Research Engine is a peer-reviewed journal and social network to connect academia. We are adding peer review to dexs. This platform requires verified .edu emails to register.
